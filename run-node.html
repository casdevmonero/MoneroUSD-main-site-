Run Your Own Node

Set up a MoneroUSD node and mine USDm to your own wallet. Mining rewards go only to the node owner.

Detected OS: Detecting... — commands below adapt to your system.

1. Install Dependencies

macOS (Homebrew)

# Install Homebrew if needed
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install build tools
brew upgrade cmake && brew upgrade make
brew install boost openssl hidapi zmq libsodium miniupnpc readline protobuf

# Or install all deps at once
brew bundle --file=MoneroUSD-main/contrib/brew/Brewfile

Ubuntu / Debian

sudo apt update && sudo apt install -y build-essential cmake pkg-config \
  libboost-all-dev libssl-dev libzmq3-dev libsodium-dev libhidapi-dev \
  libreadline-dev libminiupnpc-dev libunbound-dev libprotobuf-dev \
  protobuf-compiler ccache

2. Clone and Build

# Clone the repository
git clone https://github.com/monerousd/MoneroUSD-main.git
cd MoneroUSD-main

# Build (auto-detects CPU cores)
make release -j$(nproc 2>/dev/null || sysctl -n hw.ncpu 2>/dev/null || echo 2)

3. Set OpenSSL Path (macOS only)

export OPENSSL_ROOT_DIR="$(brew --prefix openssl)"

4. Start the Daemon

# Create data directory
mkdir -p ~/.monerousd

# Start the daemon
./build/release/bin/USDmd \
  --data-dir ~/.monerousd \
  --detach \
  --log-file ~/.monerousd/USDmd.log \
  --pidfile ~/.monerousd/USDmd.pid

# Verify it's running
kill -0 $(cat ~/.monerousd/USDmd.pid) && echo "Daemon running"

First sync may take time depending on your connection speed.

5. Create a Wallet

# Generate a new wallet (save your seed phrase!)
./build/release/bin/monerousd-wallet \
  --daemon-address 127.0.0.1:17750 \
  --generate-new-wallet ~/monerousd-wallet/miner \
  --password "YOUR_SECURE_PASSWORD"
Important: Write down your 25-word seed phrase and store it safely. It is the only way to recover your wallet.

6. Start Mining

# Get your wallet address
./build/release/bin/monerousd-wallet \
  --daemon-address 127.0.0.1:17750 \
  --wallet-file ~/monerousd-wallet/miner \
  --password "YOUR_SECURE_PASSWORD" \
  --command address

# Set your miner address and start the desktop app
export MINER_ADDRESS="YOUR_WALLET_ADDRESS_HERE"
node server.js

# Or mine directly from the wallet CLI
# In the wallet prompt, type: start_mining 1

Mining rewards go only to the MINER_ADDRESS you set. Other users cannot redirect rewards.

7. Run Swap Service (Optional)

cd swap-service && npm install

export USDM_WALLET_RPC_URL=http://127.0.0.1:27750
export BTC_NETWORK=mainnet
export MINER_ADDRESS="YOUR_WALLET_ADDRESS"
node index.js
# Swap backend running on http://0.0.0.0:8787

8. Verify Your Node

# Check daemon status
curl -s http://127.0.0.1:17750/json_rpc \
  -d '{"jsonrpc":"2.0","id":"0","method":"get_info"}' \
  -H "Content-Type: application/json" | python3 -m json.tool

# Check mining status
curl -s http://127.0.0.1:17750/mining_status \
  -H "Content-Type: application/json" | python3 -m json.tool
